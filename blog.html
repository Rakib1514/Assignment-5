1- What is the Document Object Model (DOM)?

The DOM (Document Object Model) is a structured representation of HTML. It allows us to interact with every element on a webpage. With the DOM, we can access, modify, add, or delete elements, change styles dynamically, handle events like clicks, and update content without reloading the page. This makes it essential for creating interactive web applications.

2- How do you select an element from the DOM?

With the DOM, we can select elements using a few methods built into JavaScript. The methods are getElementById, getElementsByClassName, querySelector, and querySelectorAll.
Example:
const btn = document.getElementById('id-name');

3- What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the context of the DOM where a single event listener is added to a parent element instead of individual listeners for each child element. This approach is useful for several reasons. First, it enhances performance by reducing the number of event listeners in the DOM, which is especially beneficial when dealing with many child elements. Second, it simplifies handling events for dynamically added child elements, as they automatically inherit the event listener from the parent without needing to reattach it. Additionally, this technique leads to cleaner and more manageable code, as you only need to define one listener for a group of elements. Finally, it leverages event bubbling, where events propagate up from the target element to its ancestors, allowing the parent to efficiently handle events triggered by its children.

4- How do you manipulate an element's attributes and styles using the DOM?

We can manipulate an element's attributes and styles in the DOM using a few straightforward methods. For attributes, we can set, get, or remove values with setAttribute, getAttribute, and removeAttribute. When it comes to styles, we can directly change inline styles through the style property or manage CSS classes with classList to add, remove, or toggle them. These techniques enable us to create dynamic and interactive web applications.
